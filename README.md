# PRODIGY_GA_04

## Image-to-Image Translation using Pix2Pix (cGAN)

### Objective
Implement an image-to-image translation model using a Conditional Generative Adversarial Network (cGAN) called Pix2Pix.

### Technologies Used
- Python
- TensorFlow
- Keras
- Pix2Pix
- NumPy
- Matplotlib

### Dataset
CMP Facade Database (Facades Dataset)

### Features
- Conditional GAN (cGAN)
- U-Net Generator
- PatchGAN Discriminator
- Image-to-Image Translation

### Results
The model was trained to translate semantic building layouts into realistic building facade images using the Pix2Pix architecture.
